# Media Fixtures

These files are user-provided, rights-clean fixtures for Otos tests. They are
small enough to keep in the repository and cover early metadata edge cases.

Files:

- `jpeg-exif-offset-date.jpg`: JPEG with EXIF capture date and timezone offset
  behavior.
- `ios-heic-offset-date.heic`: HEIC from iOS with offset-date behavior.
- `mp4-no-embedded-capture-date.mp4`: MP4 without embedded capture date.
- `mov-no-embedded-capture-date.mov`: QuickTime movie without embedded capture
  date.

Privacy rules:

- Do not add personal media casually.
- Do not add files without clear rights to commit them.
- Do not add fixtures containing sensitive subjects, locations, or private
  metadata.
- Do not log fixture filenames as a pattern for production media logging.
